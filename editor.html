Designed scope (workflow: always start from an existing SVG):

  • Select / move / transform paths
  • Delete paths you don't want plotted
  • Layer assignment (separate strokes by color for multi-pen jobs)
  • Stroke order / reverse direction (control plot sequence)
  • Hatch tool — convert filled shapes to hatched line patterns
  • Optimize selected (linemerge / linesort / linesimplify on selection)
  • Pan / zoom
